# PipePilot AI — Module 1

This repository contains the first module of the PipePilot AI plumbing receptionist demo.

## What is included

- Next.js 15 app router setup
- TailwindCSS styling
- Landing page for a demo plumbing company
- About, Services, Contact pages
- Floating AI chat widget placeholder
- Clean folder structure for future backend, Supabase, and OpenAI integration

## Install

1. Install dependencies:

```bash
npm install
```

2. Run the app locally:

```bash
npm run dev
```

3. Open http://localhost:3000
